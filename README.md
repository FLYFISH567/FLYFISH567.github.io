# FLYFISHING

我的个人博客仓库：记录学习、生活与持续成长。

在线访问：<https://flyfish567.github.io/>

## 博客简介

这是一个基于 **Hexo + Anzhiyu** 搭建的个人博客，主要用于整理我的学习笔记、CTF / Web 安全相关内容，以及大学生活中的一些记录与思考。

博客名称为 **FLYFISHING**，站点内容目前涵盖文章归档、分类、标签、留言板、音乐馆、追番页、相册集和关于页等多个页面，既是学习输出的地方，也是个人记录的小站。

## 内容方向

目前博客的内容主要集中在以下几个方向：

- **Web 安全学习记录**：如 SQL 注入、PHP 反序列化、SSRF、SSTI、XSS、文件上传、命令执行、目录遍历等
- **CTF / WP 题解整理**：记录做题过程、知识点和复盘思路
- **靶场搭建与环境配置**：如 XSS-labs、命令执行靶场等环境搭建过程
- **大学与生活记录**：学习过程、日常想法、个人成长与兴趣内容

## 仓库用途

这个仓库主要用于：

- 存放博客源码与配置文件
- 管理文章内容与静态资源
- 进行本地编写、生成与部署
- 同步 GitHub Pages 页面

## 本地使用

安装依赖后可使用以下命令：

```bash
npm install
hexo clean
hexo g
hexo s
```
部署到 GitHub Pages：
```
hexo clean && hexo g && hexo d
```
