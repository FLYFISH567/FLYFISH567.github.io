如何制作一个博客

Created2024-10-29|Updated2024-10-30
|Post Views:

工具准备

1.Node.js

2.git

3.Hexo

4.文本编辑器(推荐VSCODE)

5.GitHub 帐号

安装hexo

在桌面右键选择git bash，输入npm install -g hexo-cli

注册仓库

  1. 点击new直接创建仓库

  1. 命名仓库：用户名+.github.io
  2. 勾选 Initialize this repository with a README 初始化一个 README.md 文件
  3. 点击 Creat repository 进行创建

git初始化

打开git bash

通过<font style="color:rgb(244, 116, 102);">git config -l</font> 检查是否配置成功。

连接github

  1. 生成ssh公钥

在git中执行ssh-keygen -t rsa -C "你的邮箱"

之后打开C盘下用户文件夹下的.ssh的文件夹，会看到 id_rsa.pub

  1. 打开 id_rsa.pub，复制其中的内容
  2. 将 SSH KEY 配置到 GitHub

进入github，点击右上角头像 选择<font style="color:rgb(244, 116, 102);">settings</font>，进入设置页后选择 <font style="color:rgb(244, 116, 102);">SSH and GPG keys</font>

title可以随便填，将密钥填入key中，点击add ssh key即可

4.测试

在git中输入ssh -T git@github.com，若有账户信息返回则连接完成

初始化Hexo项目

  1. 选择一个文件夹，右键打开git bash 输入hexo init

  1. 再输入hexo s 则运行可以在本地浏览器输入localhost:4000打开

将静态博客挂载到 GitHub Pages
